Registrador de Filmes e Livros
O Registrador de Filmes e Livros é um sistema voltado à organização e catalogação de mídias audiovisuais e literárias, oferecendo ao usuário uma forma centralizada de gerenciar seus conteúdos pessoais.
Descrição Geral
O sistema permite que usuários criem e gerenciem listas personalizadas de livros e filmes, funcionando como uma estante virtual.
Essas listas podem conter diferentes categorias definidas pelo próprio usuário, como por exemplo: “Lidos/Assistidos em 2025”.
Funcionalidades Principais
•	Autenticação de Usuário: O acesso ao sistema é restrito a usuários cadastrados, mediante login e senha.
•	Gerenciamento de Listas: O usuário pode criar listas de mídias, adicionando ou removendo itens conforme desejar. Também é possível manter listas vazias.
•	Sistema de Resenhas e Avaliações: O usuário pode registrar comentários sobre as mídias e atribuir uma avaliação em estrelas (de 1 a 5).
•	Registro de Datas: Cada mídia adicionada à lista poderá conter o registro da data de leitura (para livros) ou da data em que o filme foi assistido.
•	Sugestões de Mídias: Caso a mídia desejada não esteja disponível, o usuário poderá sugeri-la por meio de um formulário.
o	As sugestões são enviadas ao administrador, responsável por validar as informações e decidir sobre a inclusão da mídia no banco de dados.
•	Área de Sugestões e Destaques: Após o login, o sistema exibirá uma aba inicial com recomendações e sugestões de livros e filmes.
Perfil de Usuário
•	Usuário Comum: Pode criar listas, adicionar mídias existentes, registrar resenhas, atribuir avaliações em estrelas, registrar datas e enviar sugestões.
•	Administrador: Possui privilégios de adicionar ou remover mídias do banco de dados e validar sugestões enviadas pelos usuários.
